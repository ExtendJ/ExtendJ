# ExtendJ Ant Task

**Status:** wontfix

It would be convenient to have an Ant task for ExtendJ. This would make it easier to use ExtendJ in Ant builds.

## Comments

### Jesper Öqvist - 2017-12-13

An ExtendJ Ant task could be developed as a separate project that depends on ExtendJ. I want to avoid having a dependency on Ant for this feature in the main ExtendJ build.
